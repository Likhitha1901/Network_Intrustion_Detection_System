# Network_Intrustion_Detection_System
Network Intrusion Detection System (NIDS) monitors network traffic to detect malicious activities and unauthorized access. It uses machine learning algorithms like Decision Tree and Random Forest to classify traffic as normal or suspicious. The system generates alerts for potential threats, improving real-time network security.
📌 Network Intrusion Detection System (NIDS)
📖 Overview
This project focuses on developing a Network Intrusion Detection System (NIDS) that monitors network traffic and detects malicious activities such as unauthorized access, attacks, and anomalies. The system uses machine learning techniques to classify network traffic as normal or intrusive.
🎯 Objectives
Detect suspicious and malicious network activities
Classify traffic into normal or attack categories
Improve network security using intelligent models
Reduce false positives and enhance detection accuracy
⚙️ Technologies Used
Python
Machine Learning Algorithms (e.g., Random Forest, Decision Tree, Logistic Regression)
Pandas, NumPy
Scikit-learn
Matplotlib / Seaborn (for visualization)
📊 Dataset
The dataset used for this project contains labeled network traffic data.
It includes features such as protocol type, service, flag, source bytes, destination bytes, etc.
The dataset size is approximately 15 MB (15,021 KB).
It consists of both normal traffic and various types of network attacks.
🔍 Features
Data preprocessing and cleaning
Feature selection and extraction
Model training and testing
Performance evaluation using metrics like accuracy, precision, recall, and F1-score
Visualization of results
🚀 How It Works
Load and preprocess the dataset
Split data into training and testing sets
Train machine learning models
Evaluate model performance
Predict whether network activity is normal or malicious
📈 Results
The system achieves good accuracy in detecting intrusions
Effectively distinguishes between normal and attack traffic
Provides reliable performance for network monitoring
